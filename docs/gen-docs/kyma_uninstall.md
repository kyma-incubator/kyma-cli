## kyma uninstall

Uninstalls Kyma from a running Kubernetes cluster.

### Synopsis

Uninstalls Kyma from a running Kubernetes cluster.

Make sure that your KUBECONFIG is already pointing to the target cluster.<br>
This command:
- Removes your cluster administrator account.
- Removes Tiller.
- Removes Kyma Installer.
